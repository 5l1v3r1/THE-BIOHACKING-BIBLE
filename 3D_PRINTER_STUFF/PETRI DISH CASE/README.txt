# Summary

Carrying case for Petri dishes. Holds four Petri dishes, is kept closed by 5x5x5 cubic neodymium magnets fixed in grooves of superglue.

# Print Settings

Printer: Zonestar P802N
Rafts: No
Supports: No
Resolution: 0.2
Infill: 20%

Notes: 
In general, it holds without magnets, but with them is more reliable. When installing the magnets, you should not be wrong with the polarity.
